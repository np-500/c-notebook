# Crypto Dashboard

Multi-signal crypto prediction notebook (Binance, 4h candles, 3-10 day horizon).

## Setup
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## Usage
1. Run all cells top to bottom (Sections 1-10)
2. Run Section 14 for market headlines
3. Run Section 15 for copy-to-clipboard summary
